This plugin is a widget to customize and add more css options to the Default Wordpress Category Widget. This plugin requires fontawesome in your theme to work correctly.

== Description ==

If you need to add more css and customization options to the Default Wordpress Category Widget this plugin is for you.
Using this plugin you can display your sidebar categories in a tree-view in just one click.

= Features =
*   The selection state is preserved (the parent category remain open)
*   Fully customized and easy to use
*   Display as dropdown or tree-view
*   Dynamically create your website tree for all posts & categories
*   Ability to hide/show count
*   Ability to hide/show child
*   When a child is selected The parent category is also selected


== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'art-post-categories-tree.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard
1. Place the 'Display Categories Tree' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

= Using FTP =

1. Extract 'art-post-categories-tree.zip' to your computer
1. Upload the 'art-post-categories-tree' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Place the 'Display Categories Tree' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
